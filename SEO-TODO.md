# SEO/GEO Remaining Tasks

## P2 — High Impact

### Add Statistics & Citations to Page Content
- Rewrite page copy to include specific numbers and authoritative sources
- Princeton GEO research shows +37% boost from statistics, +40% from citations
- Example: "Serving 500+ Winnipeg homes" or citing cleaning industry standards
- Apply to homepage, services page, and about page

### Submit to Brave Search
- Go to https://search.brave.com/webmasters
- Submit https://www.spongeprocleaningservices.com
- Claude AI uses Brave Search (not Google) for citations

### Google Search Console Verification
- `src/app/layout.tsx` line 64 has placeholder `'your-google-verification-code'`
- Set up at https://search.google.com/search-console
- Replace placeholder with actual verification code

### Bing Webmaster Tools
- Submit site at https://www.bing.com/webmasters
- Required for Microsoft Copilot AI visibility
- Copilot pulls exclusively from Bing index

## P3 — Long-Term

### Blog / Resources Section
- Build topical authority with content like:
  - "Move-Out Cleaning Checklist for Winnipeg Renters"
  - "How Often Should You Deep Clean Your Home?"
  - "Standard Cleaning vs Deep Cleaning: What's the Difference?"
  - "Winnipeg Spring Cleaning Guide"
- Each post should include FAQ schema, statistics, and internal links to services
- Targets long-tail keywords that AI search engines cite heavily
